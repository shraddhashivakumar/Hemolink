# Hemolink

Hemolink is a web-based platform designed to connect blood donors and recipients efficiently. It enables users to register, request blood, and locate hospitals through an easy-to-use interface.

---

## 🚀 Getting Started

### 🔗 Clone the Repository

```bash
git clone https://github.com/shraddhashivakumar/Hemolink.git
cd Hemolink
```

---

## 🛠️ Installation

### Backend Setup

```bash
cd backend
npm install
```

### Frontend Setup

No installation required for frontend (static files).

---

## ▶️ Running the Project

### Start Backend Server

```bash
cd backend
node server.js
```

### Open Frontend

Open `frontend/index.html` in your browser.

---

## 📌 Features

* Blood donor registration system
* Blood request submission
* Hospital information display
* Public map interface
* FAQ section for user guidance
